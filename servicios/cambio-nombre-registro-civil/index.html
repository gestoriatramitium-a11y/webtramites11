Registro Civil

Cambio de Nombre

El cambio de nombre permite modificar oficialmente el nombre cuando se cumplen los requisitos legales y existe justificación suficiente.

Solicitar servicio Ver todos los servicios

Para qué sirve

  • Uso habitual de un nombre distinto.
  • Corrección, adaptación lingüística o razones justificadas.
  • Orden, forma o modificación relacionada con datos registrales.

Documentación o datos necesarios

  • DNI, NIE o pasaporte.
  • Certificado de nacimiento.
  • Pruebas o documentos que justifiquen el cambio solicitado.

Cómo lo gestiona Tramitium

  • Estudiamos la viabilidad del caso.
  • Ordenamos la documentación justificativa.
  • Preparamos la solicitud para presentarla ante el Registro Civil.

Preguntas frecuentes

¿Siempre se concede?

No. Deben cumplirse requisitos legales y acreditar una razón suficiente.

¿Puedo cambiar apellidos?

Los cambios de apellidos tienen reglas específicas y deben analizarse aparte.

¿Conviene justificar el uso habitual?

Sí. Cuanta más prueba exista, mejor puede prepararse el expediente.
